Qu'est-ce que le JTBD ?

Le framework Jobs-to-be-Done part d'un principe fondamental : les utilisateurs n'achètent pas des produits, ils "embauchent" des solutions pour accomplir des tâches précises dans leur vie.

Comprendre ces jobs profonds — fonctionnels, sociaux, émotionnels — permet de créer des produits qui résonnent vraiment avec les besoins des utilisateurs.

Les 3 types de jobs

Jobs fonctionnels : Ce que l'utilisateur cherche à accomplir concrètement. Jobs sociaux : Comment il veut être perçu par les autres. Jobs émotionnels : Comment il veut se sentir.
